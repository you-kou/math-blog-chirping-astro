## 标准定义

标准形式的指数函数表达式为：

$$f(x) = a^x \quad (a > 0 \text{ 且 } a \neq 1)$$

- **底数 $a$（Base）：** 必须大于 0 且不等于 1。
  - 若 $a \le 0$，在实数范围内当 $x$ 为分数时（如 $(-2)^{1/2}$）会产生虚数，导致函数不连续、无意义。
  - 若 $a = 1$，$1^x$ 恒等于 1，函数退化为常数函数，失去指数函数的动态增长特征。
- **自变量 $x$（Exponent）：** 定义域为全体实数 $\mathbb{R}$，即 $x \in (-\infty, +\infty)$。
- **因变量 $f(x)$：** 值域为 $(0, +\infty)$。图像永远在 $x$ 轴上方，且以 $x$ 轴为渐近线。

## 核心图像与性质

根据底数 $a$ 的范围，指数函数的形态分为以下两种截然相反的走向：

### 指数衰减（0 < a < 1）

- **单调性：** 在定义域 $(-\infty, +\infty)$ 内**严格单调递减**。
- **变化趋势：**
  - 当 $x \to +\infty$ 时，$y \to 0$（$x$ 轴正半轴为水平渐近线）。
  - 当 $x \to -\infty$ 时，$y \to +\infty$。
- **典型实例：** 放射性元素的衰变、半衰期计算。

![放射性元素衰变](<./assets/指数函数 - 放射性元素衰变.png>)

### 指数增长（a > 1）

- **单调性：** 在定义域 $(-\infty, +\infty)$ 内**严格单调递增**。
- **变化趋势：**
  - 当 $x \to +\infty$ 时，$y \to +\infty$（呈“爆炸式”增长）。
  - 当 $x \to -\infty$ 时，$y \to 0$（$x$ 轴负半轴为水平渐近线）。
- **典型实例：** 细胞分裂、无资源限制的人口增长、复利计算。

![复利](<./assets/指数函数 - 复利.png>)

> 💡 **单调性应用：**
> 在解指数不等式时：
>
> - 若 $a > 1$，则 $a^{x_1} > a^{x_2} \iff x_1 > x_2$ （不等号方向不变）；
> - 若 $0 < a < 1$，则 $a^{x_1} > a^{x_2} \iff x_1 < x_2$ （不等号方向改变）。

### 共同几何特征

无论 $a$ 取何值，图像均具有以下特征：

- **必过定点 $(0, 1)$：** 因为对任意非零实数，都有 $a^0 = 1$。
- **必过定点 $(1, a)$：** 因为 $a^1 = a$。

## 核心运算法则

设 $x, y$ 为任意实数，以下法则是化简求导、求极限及解方程的基础：

| 法则名称         | 数学公式                                       | 直观理解与记忆口诀                 |
| :--------------- | :--------------------------------------------- | :--------------------------------- |
| **同底数幂相乘** | $a^x \cdot a^y = a^{x+y}$                      | 底数相同，乘法化为**指数相加**     |
| **同底数幂相除** | $\frac{a^x}{a^y} = a^{x-y}$                    | 底数相同，除法化为**指数相减**     |
| **幂的乘方**     | $(a^x)^y = a^{xy}$                             | 幂的乘方，化为**指数相乘**         |
| **积的乘方**     | $(ab)^x = a^x \cdot b^x$                       | 积的乘方，等于各因式分别乘方后相乘 |
| **商的乘方**     | $\left(\frac{a}{b}\right)^x = \frac{a^x}{b^x}$ | 分式乘方，等于分子分母分别乘方     |

## 特殊指数与负指数法则

在处理极限与导数计算时，需熟练将负指数和分数指数进行恒等变形：

- **零指数幂：**
  $$a^0 = 1 \quad (a \neq 0)$$
  _证明：_ $\frac{a^x}{a^x} = a^{x-x} = a^0 = 1$。
- **负指数幂（倒数关系）：**
  $$a^{-x} = \frac{1}{a^x}$$
  _示例：_ $2^{-3} = \frac{1}{2^3} = \frac{1}{8}$。
- **分数指数（根式互化）：**
  $$a^{\frac{m}{n}} = \sqrt[n]{a^m} \quad (a > 0, n > 1, n \in \mathbb{N}^*)$$
  _示例：_ $4^{\frac{3}{2}} = \sqrt{4^3} = \sqrt{64} = 8$。
